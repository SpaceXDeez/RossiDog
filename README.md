# RossiDog
RossiDog home page

**Hello World Website**

You can run a tiny local server from the project root to view the demo site.

Run:

```bash
cd /workspaces/RossiDog
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser. From the host environment you can also run:

```bash
$BROWSER http://localhost:8000
```

Files added:
- `index.html` — main page
- `styles.css` — stylesheet
- `script.js` — small demo script
